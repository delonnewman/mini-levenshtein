# mini-levenshtein

Simple, fast, levenshtein distance

# Synopsis

```ruby
MiniLevenshtein.edit_distance("levenshtein", "levenstien") # => 3

MiniLevenshtein.similarity("levenshtein", "levenstien") # 0.857142...
```
